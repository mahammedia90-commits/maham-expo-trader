## المرحلة 2: صفحة اكتشاف المعارض (BrowseExpos)
- [ ] إضافة فلاتر متقدمة: دولة، قطاع، نطاق سعر، حجم البوث
- [ ] إضافة تصنيفات القطاعات: تقنية، تعليم، جمال، صحة، أغذية، بناء، رياضة، سيارات
- [ ] إضافة عنصر Scarcity: "باقي 3 أجنحة فقط"
- [ ] إضافة Live Activity: "شركة من دبي حجزت جناح قبل 5 دقائق"
- [ ] إضافة Early Booking Incentive badge

## المرحلة 3: صفحة المعرض (ExpoDetail)

- [ ] إضافة الرعايات المتاحة (Gold, Silver, Startup, Innovation)
- [ ] إضافة عناصر التحفيز النفسي (Scarcity, Price Alert)
- [ ] إضافة Live Activity notifications

## المرحلة 4: نظام الحجز المحسّن
- [ ] إنشاء سلة حجز ظاهرة (Booking Cart) في الهيدر
- [ ] تحسين ملخص الدفع (سعر + رسوم تشغيلية + ضريبة)
- [ ] إضافة Price Alert في صفحة الحجز

## المرحلة 5: الخدمات التشغيلية (Operations Marketplace)
- [ ] إعادة بناء صفحة Operations كـ Marketplace
- [ ] إضافة خدمات: تصميم البوث، كهرباء، إنترنت، لوجستيات، طباعة، إدارة مخزون
- [ ] نماذج طلب مع تحديد الكمية والموقع والتاريخ
- [ ] نظام تتبع حالة الطلب (قيد المراجعة، قيد التنفيذ، مكتمل)

## المرحلة 6: نظام CRM Pipeline

## المرحلة 7: لوحة التحليلات
